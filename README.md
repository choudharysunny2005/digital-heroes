# Digital Heroes eCommerce Platform

A premium, modern eCommerce platform featuring glassmorphism design, intelligent Amazon-style search engine, and an extensive admin dashboard.

## Features
- 🚀 Ultra-fast load times with lazy loaded images and deferred scripts
- 🔍 Amazon/Flipkart style Mega Search Bar with real-time suggestions
- 💳 Cart and Checkout capabilities
- 📊 Fully functional Admin Dashboard
- 🌙 Dynamic Dark/Light mode support
